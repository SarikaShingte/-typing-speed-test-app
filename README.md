# ⌨️ Typing Speed Test App

## 📌 About Project
This is a simple web-based application that measures a user's typing speed and accuracy in real time.

---

## 🚀 Features
- Real-time Words Per Minute (WPM) calculation
- Accuracy percentage calculation
- Timer-based typing test
- Random sentence generation
- Restart test option

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎯 How It Works
1. Click the Start button
2. Timer begins
3. Type the given sentence
4. System calculates WPM and accuracy
5. Final result is displayed
